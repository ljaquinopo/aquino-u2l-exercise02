Project 1: Blooms && Bees

Honey bees will collect blooms (flowers) that will be converted into honey (points), the game will continue until the bee dies (from a collision with a praying mantis, dragonflies, ambush bugs, or webs) or reach the checkpoint.

Project 2: Weather Portal

This website provides users with the ability to input a city from the United States and receive the current weather conditions for that location. API Used: openweathermap.org

Project 3: Unbeatable Tic-Tac-Toe

Utilized an advanced AI technique known as the Minimax algorithm with alpha-beta pruning, which ensures the AI opponent makes strategic moves, resulting in a game where winning is highly challenging or virtually impossible for the player.

Project 4: Machine Learning Algorithms
